# Event Registration System

This is a full-stack web application that allows users to register for events and helps organizers manage participant details easily.

Frontend: React, HTML, CSS, JavaScript  
Backend: Node.js, Express.js  
Database: MongoDB  
Version Control: Git & GitHub
